> *"The AI That Never Forgets You"*

**Mnemo AI** is a complete Life Intelligence Platform that builds a permanent knowledge graph of your entire life journey — every struggle, every win, every dream.

---

## 🎯 Hackathon Track

**Track 1: Best Use of Cognee Open Source**

- ✅ Uses **Cognee V1 Open Source**
- ✅ Implements all **4 core Cognee APIs**: `remember`, `recall`, `improve`, `forget`
- ✅ Built with **₹0 budget**
- ✅ **Solo developer** — 7 days

---

## ✨ Features

- 🌙 **DayEnd Ritual** — Journal with emotion detection
- 🧠 **Life Ask** — Chat with memory recall
- 📈 **Insights** — Mnemo Score & Memory DNA
- 🎯 **Goals** — Lock in your goals
- ⏳ **Timeline** — Visual life journey
- 💌 **Future Self** — Time capsules
- 🗑️ **Forget & Heal** — Remove memories
- 🎁 **Wrapped** — Weekly life summary
- 🎤 **Voice Input/Output** — Web Speech API
- 🌓 **Dark/Light Mode** — Theme switching
- ⌨️ **Keyboard Shortcuts** — Ctrl+J, Ctrl+C, etc.
- 📱 **PWA** — Installable as mobile app
